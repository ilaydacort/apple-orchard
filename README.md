# 🍎 Apple Orchard

Python ve Pygame ile geliştirilmiş pixel art tarzı kaçış oyunu.

## Hikaye

Elma bahçesinde elmalar üstüne yağıyor! Kapşonlu karakterini kontrol et, düşen elmalardan kaç ve hayatta kal. 🌳

## Özellikler

- Pixel art tarzı grafikler
- Artan zorluk sistemi (her 5 saniyede hız artar)
- Skor takibi
- SPACE ile sprint özelliği
- R tuşu ile yeniden başlama

## Kurulum

### Gereksinimler

- Python 3.x
- Pygame

### Pygame Kurulumu

```bash
pip install pygame
```

### Oyunu Çalıştırma

```bash
python elmadan_kacis.py
```

## Kontroller

| Tuş | İşlev |
|-----|-------|
| ← Sol ok | Sola git |
| → Sağ ok | Sağa git |
| ↑ Yukarı ok | Yukarı git |
| ↓ Aşağı ok | Aşağı git |
| SPACE | Sprint (2x hız) |
| R | Yeniden başla |
